`waiind` is an R package (repository `wai_ind_package`). This repository is
set up with the standard structure for R package development.

> **Note:** the package is named `waiind` rather than `wai_ind_package`
> because R package names may only contain letters, numbers and dots — they
> cannot contain underscores.

## Installation

You can install the development version of `waiind` from GitHub with:

``` r
# install.packages("devtools")
devtools::install_github("PhilippKronenberg/wai_ind_package")
```

## Example

``` r
library(waiind)

hello_world()
#> [1] "Hello, world!"

hello_world("Philipp")
#> [1] "Hello, Philipp!"
```

## Development workflow

This package uses [roxygen2](https://roxygen2.r-lib.org/) for documentation
and [testthat](https://testthat.r-lib.org/) for testing. Common commands
(run from the package root):

``` r
# install.packages("devtools")

devtools::document()   # Regenerate NAMESPACE and man/ from roxygen comments
devtools::test()       # Run the test suite
devtools::check()      # Run R CMD check
devtools::load_all()   # Load the package for interactive development
```
